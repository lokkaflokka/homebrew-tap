# Homebrew Tap

Custom Homebrew formulas.

## Install

```bash
brew tap lokkaflokka/tap
```

## Formulas

| Formula | Description |
|---------|-------------|
| [eventkit-cli](https://github.com/lokkaflokka/eventkit-cli) | CLI for Apple Reminders and Calendar via EventKit |

```bash
brew install lokkaflokka/tap/eventkit-cli
```
